# react-admin-dva

工作的后台模板

## 环境

编辑器使用 ```vscode``` ，依赖的安装建议使用 ```yarn``` 进行，```npm 5.x``` 安装某些依赖时会出错。

* ```node v8.5.0```
* ```react v16.2.0```
* ```antd v3.1.1```
* ```dva v.2.1.0```


## 如何使用

```
npm run start
```

## 代码风格

```js``` 代码使用 ```dva``` 框架默认的 ```eslint``` 配置，```css``` 代码使用 ```csscomb``` 插件进行优化。
